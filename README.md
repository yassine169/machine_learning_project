# Projet Machine Learning - Classification du Risque de Crédit

## Description

Ce projet a été réalisé dans le cadre d’un travail universitaire.
L’objectif est de prédire le risque de crédit d’un client en utilisant plusieurs modèles de Machine Learning.

En plus de la modélisation, une approche simple de MLOps a été mise en place avec MLflow, ainsi qu’une application web permettant d’interagir avec les modèles.

---

## Objectifs

* Tester plusieurs modèles de classification : KNN, SVM, Random Forest et Régression Logistique
* Comparer leurs performances
* Suivre les expérimentations avec MLflow
* Développer une application avec un backend et un frontend

---

## Technologies utilisées

* Python (Scikit-learn, Pandas, NumPy)
* MLflow
* FastAPI
* React et TailwindCSS
